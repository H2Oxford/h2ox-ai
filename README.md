# h2ox-ai
This repo is for training machine learning models for the Wave2Web hackathon.


## Installation

### For use

    pip install .

### For Development

    # install with "-e" (editable) and [dev] flag to get pre-commit
    pip install -e .[dev]

    # install pre-commit
    pre-commit install


## Training

    python run.py


## Serving

    app.py
